## Using Each Feature

### How to Swap Tokens

1. Navigate to **Swap** page
2. Select input token (e.g., SOL)
3. Enter amount to swap
4. Select output token (e.g., USDC)
5. Review estimated output (auto-updates)
6. Set slippage tolerance
7. Click **"Swap Tokens"**
8. Confirm transaction in wallet
9. Wait for confirmation (5-30 seconds)

**Pro Tip**: Lower slippage = lower chance of success but better price

### How to Use Arbitrage

1. Navigate to **Arbitrage** page
2. Review flash loan providers and fees
3. Set minimum profit threshold (default: 0.5%)
4. Toggle auto-execute ON/OFF
5. Click **"Start Scanning"**
6. Monitor live opportunities feed
7. Click **"Execute"** on profitable opportunities
8. Confirm transaction in wallet

**Pro Tip**: Start with higher profit thresholds (1%+) to filter noise

### How to Use Sniper Bot

1. Navigate to **Sniper** page
2. Configure settings:
   - Buy amount (recommended: 0.1-1 SOL)
   - Slippage tolerance (recommended: 10-20%)
3. Toggle auto-snipe ON/OFF
4. Click **"Start Monitoring"**
5. Watch detected targets appear
6. Click **"SNIPE NOW"** to purchase
7. Confirm transaction in wallet

**Warning**: Sniping carries high risk. Only invest what you can afford to lose.

### How to Launch a Token

1. Navigate to **Launchpad** page
2. Fill in token details:
   - Token Name
   - Token Symbol (max 10 characters)
   - Total Supply
3. Set airdrop percentage (1-50%)
4. Review deployment cost
5. Click **"Deploy Token"**
6. Confirm transaction in wallet
7. Try the airdrop roulette after deployment

### How to Check Airdrops

1. Navigate to **Airdrop** page
2. Connect wallet (auto-scans after connection)
3. Review wallet score and tier
4. Check claimable airdrops list
5. Click **"Claim"** on individual airdrops
6. Or click **"Claim All"** for batch claiming
7. Confirm transactions in wallet

### How to Stake SOL

1. Navigate to **Staking** page
2. Review portfolio tracker (if you have staked assets)
3. Browse available pools
4. Click on desired pool to select
5. Enter stake amount or use quick buttons (1, 5, 10, 50 SOL)
6. Review estimated returns (1-year projection)
7. Click **"Stake Now"**
8. Confirm transaction in wallet

**Pro Tip**: Jito and Kamino offer higher APY but may have different risk profiles

### How to Analyze Wallets

1. Navigate to **Wallet Analysis** page
2. Enter Solana wallet address
3. Click **"Analyze Wallet"**
4. Wait 5-30 seconds for analysis
5. Review results:
   - Risk assessment score and level
   - Wallet metadata (age, balance, transactions)

6. Use insights for investment decisions or due diligence

## Troubleshooting

### "Transaction Failed"
**Causes**:
- Insufficient SOL for gas fees
- Slippage too low
- Network congestion
- Token liquidity issues

**Solutions**:
- Ensure 0.1+ SOL for fees
- Increase slippage tolerance
- Try again during off-peak hours
- Check token liquidity on DEX

### "Wallet Not Connecting"
**Causes**:
- Wallet extension not installed
- Wrong network selected
- Browser compatibility issues

**Solutions**:
- Install wallet extension (Phantom, Solflare)
- Select Solana Mainnet in wallet
- Try different browser (Chrome recommended)
- Clear browser cache and cookies

### "Prices Not Updating"
**Causes**:
- Internet connection issues
- RPC node problems
- Browser blocking scripts

**Solutions**:
- Check internet connection
- Refresh page (Ctrl+R / Cmd+R)
- Allow scripts in browser settings
- Try different device/browser

### "Opportunities Not Appearing"
**Causes**:
- Profit threshold too high
- Market conditions unfavorable
- Scanning not started

**Solutions**:
- Lower minimum profit threshold
- Wait for market volatility
- Ensure "Start Scanning" clicked
- Check wallet connection
