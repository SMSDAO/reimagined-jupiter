## 7. Responsive Design

All components are mobile-responsive:

### Desktop (lg):
- Full-width sliders
- Side-by-side preset buttons (grid-cols-2 or grid-cols-4)
- All controls visible

### Tablet (md):
- Slightly smaller sliders
- Wrapped preset buttons
- Compact padding

### Mobile (sm):
- Full-width stacked controls
- Larger touch targets
- Simplified layout

Example responsive classes:
```css
p-4 sm:p-6              /* Padding */
text-lg sm:text-xl      /* Font size */
grid-cols-2             /* Button grids */
gap-2 sm:gap-3         /* Spacing */
```
